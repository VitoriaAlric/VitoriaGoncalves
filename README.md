Foi incrível e me formei uma Desenvolvedora Java Jr Full Stack, atualmente estudo Analise e Desenvolvimento de Sistemas na [FIAP](https://www.fiap.com.br) e trabalho no Itaú como Desenvolvedora Back-end. 🚀
